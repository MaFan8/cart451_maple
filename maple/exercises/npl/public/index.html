Reading "Raising Happy Children for Dummies" using Natural(language processing program). The entire text was tokenized and only the verb present participle(VBG) were islolted. Each word is placed on the X-axis by it's frequency in the text, then it's font size mapped by Natural's sentiment value. (Notes: the sentiment value was multiplied by 1000 for coherency.) The 'For Dummies' books are established as instructional or reference guides. Therefore it seemed fitting that I chose to isolate the verb present participle (text that end in 'ing').
